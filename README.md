# Mindful Words - 文字摘录插件


## 🌟 项目简介

**Mindful Words** 是一款够轻松捕获、整理和管理网页上的精彩文字内容chrome浏览器扩展.

## ✨ 核心功能

### 📝 智能文字捕获
- **Ctrl+点击保存**：按住Ctrl键点击选中文本即可快速保存
- **实时反馈**：保存过程中提供清晰的状态提示（保存中/成功/失败）
- **防重复保存**：智能防抖机制，避免误操作

### 🔍 强大的笔记管理
- **实时搜索**：支持按内容、标签、来源URL进行全文搜索
- **在线编辑**：直接在弹出窗口中编辑笔记内容和标签
- **滚动记忆**：自动记住笔记列表的滚动位置
- **时间追踪**：自动记录创建和修改时间

### 🏷️ 高级标签系统
- **智能标签管理**：在选项页面统一管理所有标签
- **批量更新**：编辑标签时自动更新所有相关笔记
- **标签删除**：删除标签时自动从所有笔记中移除
- **标签搜索**：支持按标签快速筛选笔记

### 📊 多格式数据导出
- **JSON格式**：完整的结构化数据，适合程序处理
- **CSV格式**：表格数据，便于Excel等工具分析
- **TXT格式**：纯文本格式，通用性最强

### 🎨 个性化体验
- **多主题支持**：浅色、深色、跟随系统三种主题模式
- **来源追踪**：自动记录每条笔记的来源网页信息
- **响应式设计**：适配不同屏幕尺寸

## 🚀 快速开始

### 📦 安装方式

#### Chrome/Edge 浏览器
1. 下载[最新版本](https://github.com/yangyongze/mindful-words/releases/latest)源码
2. 解压到本地文件夹
3. 打开浏览器，访问 `chrome://extensions/` (Chrome) 或 `edge://extensions/` (Edge)
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"，选择解压后的项目文件夹


### 📖 使用指南

#### 🎯 基础操作
1. **快速保存文字**
   - 按住 `Ctrl` 键点击选中的文字

2. **查看笔记**
   - 点击浏览器工具栏中的 Mindful Words 图标
   - 在弹出窗口中浏览所有已保存的笔记

#### 🔍 高级功能
3. **搜索笔记**
   - 在弹出窗口顶部的搜索框中输入关键词
   - 支持搜索笔记内容、标签和来源URL
   - 点击搜索框右侧的 ✕ 按钮清除搜索

4. **编辑笔记**
   - 在笔记列表中点击"编辑"按钮
   - 在弹出的编辑窗口中修改内容和标签
   - 点击"保存"确认修改

5. **管理标签**
   - 点击笔记下方的"标签"按钮快速编辑单个笔记的标签
   - 或进入扩展选项页面进行批量标签管理

6. **导出数据**
   - 在弹出窗口中点击导出按钮
   - 选择需要的格式：JSON（结构化数据）、CSV（表格数据）、TXT（纯文本）

7. **个性化设置**
   - 右键点击扩展图标选择"选项"
   - 或在扩展管理页面点击"详细信息" → "扩展程序选项"
   - 可设置主题模式和管理标签

## 🛠️ 技术架构

### 📁 项目结构
```
mindful-words/
├── manifest.json          # 扩展配置文件 (Manifest V3)
├── background.js          # 后台服务工作者
├── content.js            # 内容脚本 (注入到网页)
├── popup.html/js         # 弹出窗口界面
├── options.html/js       # 选项页面
├── popup.css            # 样式文件
└── icons/               # 图标资源
```

### 🔧 核心技术
- **Manifest V3**：使用最新的Chrome扩展标准
- **Service Worker**：高效的后台处理机制
- **Chrome Storage API**：本地数据存储
- **Content Scripts**：网页内容交互
- **现代CSS**：响应式设计和主题支持

### 💾 数据存储
- **本地存储**：使用Chrome Storage API确保数据安全
- **数据结构**：JSON格式，支持标签、时间戳等元数据
- **备份机制**：自动重试和备用存储方案
- **数据一致性**：所有操作都确保数据完整性
