# i18n Internationalization Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Chinese/English language switching support using Chrome i18n API

**Architecture:** Create `_locales` folder with en/zh_CN translation files, extract all hardcoded text to messages.json, add language selector in settings

**Tech Stack:** Chrome Extension i18n API, JSON translation files

saveLanguagePreference() {
  const language = this.elements.languageSelect.value;
  this.data.settings.language = language;
  chrome.storage.local.set({ settings: this.data.settings }, () => {
    this.showNotification(this.i18n('languageChangeNote'), 'success');
  });
},
