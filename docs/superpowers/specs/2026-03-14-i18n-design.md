# i18n Internationalization Design

## Overview

Add Chinese/English language switching support to Mindful Words Chrome extension using Chrome i18n API.

## Requirements

- **Scope**: All extension interfaces (popup, options, content script, background)
- **Default Language**: Follow browser language (auto-detect)
- **Languages**: English (en), Simplified Chinese (zh_CN)

## Technical Approach

Use Chrome extension native `chrome.i18n` API with `_locales` folder structure.

## Implementation Details

### 1. manifest.json Changes

Add default locale:
```json
{
  "default_locale": "en"
}
```

### 2. Translation Files Format

`_locales/en/messages.json`:
```json
{
  "extName": {
    "message": "Mindful Words",
    "description": "Extension name"
  },
  "searchPlaceholder": {
    "message": "Search notes...",
    "description": "Search input placeholder"
  }
}
```

### 3. Code Usage

**In JavaScript:**
```javascript
const text = chrome.i18n.getMessage('messageKey');
```

**In HTML (via data attributes):**
```html
<button data-i18n="btnCopy"></button>
```

### 4. Translation Keys Required

| Component | Count | Examples |
|-----------|-------|----------|

| popup.js | ~20 | Toast messages, confirm dialogs |
| options.html | ~40 | Settings labels, descriptions |
| options.js | ~15 | Notifications, confirm dialogs |
| content.js | ~5 | Save success/failure messages |
| background.js | ~3 | Default note content |

## User Flow

1. **New User**: Extension auto-detects browser language
2. **Manual Switch**: User changes language in Settings > Appearance
3. **Effect**: Page refresh required for complete language change
